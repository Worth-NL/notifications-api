How to review new email branding

Make sure new email branding can remain in the branding pool. Check:

  • if it’s a duplicate
  • image quality
  • if it can be used by more than one service
  • branding name

1. Duplicates

If the same logo is in the branding pool twice then keep whichever one looks best.

2. Image quality

Check the image has a transparent background and is:

  • landscape
  • not distorted or stretched
  • not low resolution

Text in a logo

Some logos may contain text. If the logo contains the name of the service or organisation, it must be clear to read. Ideally, text size ideally should not be less than 14px.

Use your judgement for text that does not convey important information about the brand identity.

3. Used by more than one service

Logos in the pool can be used by more than one service unless a user lets us know it can’t.

4. Branding name

Users provide alt text when they upload a logo. We use this alt text to help create a name for the brand. For example:

  • Alt text provided: Greenwich Council
  • Brand name we give: Greenwich Council

Renaming branding

We can rename the branding if there are many alternate versions. Users may find it hard to choose between them if the names are not descriptive enough. For example:

  • Greenwich Council
  • Greenwich Council (alternate 1)
  • Greenwich Council (alternate 2)
  • Greenwich Council (alternate 3)
